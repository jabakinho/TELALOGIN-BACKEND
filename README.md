# Login Loja - React Native com Expo

Projeto de tela de login para a atividade de Programação Mobile.

## Como abrir no VS Code

1. Extraia o arquivo ZIP.
2. Abra a pasta no VS Code.
3. No terminal, rode:

```bash
npm install
```

4. Para executar:

```bash
npx expo start -c
```

5. Para abrir no navegador, pressione `w`.

## Tela entregue

Tela de login com campos de e-mail e senha, botão de entrar, botão do Google, recuperação de senha e cadastro.
